# Snake Water Gun Game 🎮

A fun command-line implementation of the classic Snake-Water-Gun game built with Python. The game allows the player to compete against the computer while enjoying immersive sound effects for different game actions and outcomes.

## Features

* Interactive command-line gameplay
* Random computer choices using Python's `random` module
* Sound effects using `pygame`
* Win, lose, and draw detection
* Realistic sounds for:

  * Snake hissing 🐍
  * Water splashing 💧
  * Gun firing 🔫
  * Draw situations
  * Wrong answer / losing buzzer

## Game Rules

* Snake drinks Water → Snake wins
* Water douses Gun → Water wins
* Gun kills Snake → Gun wins
* Same choices result in a Draw

## Technologies Used

* Python
* Pygame

## How to Run

1. Install Python.
2. Install Pygame:

   ```bash
   pip install pygame
   ```
3. Download the project files and sound effects.
4. Run the game:

   ```bash
   python snake_water_gun.py
   ```
